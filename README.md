# Haus Of Beauty

A modern, elegant website for Haus Of Beauty - a premium hair salon specializing in braids, twists, and wig installations.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern Animations**: Smooth GSAP animations for enhanced user experience
- **Black & White Theme**: Sophisticated monochrome color scheme
- **Service Showcase**: Comprehensive display of all hair services offered
- **Gallery Section**: Portfolio of stunning hair transformations
- **Customer Reviews**: Social proof with client testimonials

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **Vanilla CSS**: Custom styles and animations
- **JavaScript**: Interactive features
- **GSAP**: Advanced animations library
- **Lucide Icons**: Modern icon set

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ndaza02/haus-of-beauty.git
   ```

2. Open `index.html` in your browser or serve with a local server

## Deployment

### GitHub Pages
1. Go to repository Settings
2. Navigate to Pages section
3. Select `main` branch as source
4. Your site will be live at `https://ndaza02.github.io/haus-of-beauty/`

### Other Hosting
Simply upload all files to your web hosting service.
